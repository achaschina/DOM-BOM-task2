Метод event.stopPropagation()

При настанні події обробники спочатку спрацьовують на елементі на якому відбулась подія, потім на його батьківському елементі, потім вище і так далі, вгору по ланцюжку вкладеності. Остання подія вспливає на тегу body.
Якщо включити метод stopPropagation(), то всплиття зупинеться. Тобто буде спрацьовувати тільки клік на конкретному елементі.



Метод event.preventDefault()

Даний метод дозволяє виключити поведінку браузера по замовчуванню. Наприклад якщо застосувати метод preventDefault() можна відключити перехід по ссилкі.

Різниця між цими двома методами полягає у тому, що метод stopPropagation() блокує вспливання події відносно батьківського елементу. А метод preventDefault() виключає дефолтну поведінку браузера і можна перехопити і змінити подію (наприклад клік по ссилці).
